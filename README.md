# 🤖 Gemini AI Chatbot

A full-stack AI chatbot with **conversation memory** and **real-time Google Search** capability, built using Google's Gemini API and Flask.

## ✨ Features
- 🧠 Remembers conversation history
- 🌐 Real-time data via Google Search (weather, prices, news)
- 💬 Beautiful dark-themed web interface
- 📱 Fully responsive design

## 🛠️ Tech Stack
- **Backend:** Flask, Google Gemini API
- **Frontend:** HTML, CSS, JavaScript
- **Tools:** python-dotenv, Flask-CORS

## 🚀 Setup Instructions
1. Clone the repository
2. Create a `.env` file with your `GOOGLE_API_KEY`
3. Install dependencies: `pip install -r requirements.txt`
4. Run: `python app.py`
5. Open: `http://localhost:5000`
